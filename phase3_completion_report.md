## 三、功能实现详情

### 1. AI需求解析器 (`core/ai_parser.py`)

#### 核心功能
- **规则引擎**: 基于正则表达式的自然语言解析
- **参数提取**: 自动提取质控、聚类、降维、差异表达、富集分析等参数
- **参数验证**: 内置参数合法性验证
- **数据类型适配**: 支持单细胞和空间转录组数据

#### 主要类和方法
```python
class AIParser:
    - parse_requirement()          # 解析用户需求
    - validate_parameters()        # 验证参数合法性
    - _extract_qc_parameters()     # 提取质控参数
    - _extract_clustering_parameters()  # 提取聚类参数
    - _extract_enrichment_parameters()  # 提取富集分析参数
    - generate_prompt_template()   # 生成提示词模板（预留LLM接口）
```

#### 支持的自然语言指令示例
- 质控: "过滤线粒体比例超过15%的细胞"
- 聚类: "使用Leiden算法，分辨率0.8"
- 富集分析: "做GO和KEGG富集分析"
- 可视化: "生成UMAP图和火山图"

### 2. AI自然语言分析页面 (`ui/ai_analysis_page.py`)

#### 页面布局
1. **需求输入区**
   - 文本输入框：支持自定义需求
   - 示例需求选择器：提供3个预设示例
   - 复选框：是否使用示例需求

2. **解析结果预览区**
   - 质控参数卡片
   - 聚类参数卡片
   - 差异表达参数卡片
   - 富集分析参数卡片
   - 可视化配置JSON展示

3. **确认执行区**
   - "修改参数"按钮：跳转到参数配置页面
   - "一键执行分析"按钮：直接执行分析流程

4. **帮助信息区**
