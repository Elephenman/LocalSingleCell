## 快速开始

### 5分钟快速上手

```bash
# 1. 进入项目目录
cd LocalSingleCell

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动应用
streamlit run app.py --server.headless true

# 4. 浏览器打开 http://localhost:8501
```

### 第一个分析流程

1. **导入数据** → 上传 `.h5ad` 文件或点击"加载示例数据"
2. **配置参数** → 使用默认参数或自定义调整
3. **执行分析** → 点击"执行分析"按钮
4. **查看结果** → 在可视化页面查看图表
5. **导出结果** → 下载分析报告和数据

## 安装指南

### 方式一：标准安装（推荐）

```bash
# 1. 克隆或下载项目
git clone https://github.com/your-repo/LocalSingleCell.git
cd LocalSingleCell

# 2. 创建虚拟环境（推荐）
python -m venv venv

# Windows 激活
venv\Scripts\activate

# Linux/macOS 激活
source venv/bin/activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 启动应用
streamlit run app.py --server.headless true
```

### 方式二：使用启动脚本

**Windows:**
```bash
双击运行 start.bat
```

**Linux/macOS:**
```bash
chmod +x start.sh
./start.sh
```

### 方式三：Docker部署

```bash
# 构建镜像
docker build -t localsinglecell .

# 运行容器
docker run -p 8501:8501 localsinglecell
```

## 功能模块

### 1. 数据导入

#### 支持的数据格式

| 格式 | 说明 | 推荐场景 |
|-----|------|---------|
| h5ad | AnnData标准格式 | 已处理数据、标准交换格式 |
| 10x | CellRanger输出 | 新测序数据、标准流程 |
| SRA | NCBI SRA数据库 | 公共数据下载 |

#### 导入步骤

**方式一：上传h5ad文件**
1. 点击"📁 本地h5ad文件导入"
2. 拖拽或点击上传 `.h5ad` 文件
3. 等待加载完成
4. 查看数据基本信息

**方式二：上传10x格式**
1. 点击"📂 10x标准输出导入"
2. 将 `filtered_feature_bc_matrix` 文件夹压缩为zip
3. 上传zip文件
4. 系统自动识别并加载

**方式三：使用示例数据**
1. 点击"加载示例数据"按钮
2. 选择示例数据集
3. 自动加载并跳转

### 2. AI自然语言分析

#### 使用方法

1. 导入数据后，进入"🤖 AI分析"页面
2. 在文本框中输入分析需求
3. 点击"解析需求"查看参数
4. 确认后点击"一键执行分析"

#### 示例需求

```
基础分析示例：
"做一个基础的单细胞分析，用默认参数"

质控示例：
"过滤线粒体比例超过15%的细胞，最少200个基因"

聚类示例：
"用Leiden算法，分辨率0.8聚类，找每个聚类的标记基因"

富集示例：
"做GO和KEGG富集分析，生成UMAP图和气泡图"

完整流程：
"完整分析流程，过滤线粒体比例超过15%的细胞，
最少300个基因，用分辨率0.6聚类，找差异基因，
做GO、KEGG富集分析"
```

### 3. 分析流程配置

#### 质控参数

| 参数 | 默认值 | 范围 | 说明 |
|-----|-------|------|------|
| 最小基因数 | 200 | 10-1000 | 每个细胞最少检测到的基因数 |
| 最大基因数 | 6000 | 1000-20000 | 每个细胞最多检测到的基因数 |
| 最小UMI数 | 500 | 100-5000 | 每个细胞最少UMI总数 |
| 最大UMI数 | 20000 | 5000-100000 | 每个细胞最多UMI总数 |
| 最大线粒体比例 | 20% | 5-50% | 线粒体基因比例上限 |

#### 降维参数

| 参数 | 默认值 | 说明 |
|-----|-------|------|
| PCA主成分数 | 50 | PCA降维的主成分数量 |
| UMAP邻居数 | 15 | UMAP的邻居数量 |
| UMAP最小距离 | 0.5 | UMAP的最小距离参数 |

#### 聚类参数

| 参数 | 默认值 | 范围 | 说明 |
|-----|-------|------|------|
| 聚类算法 | Leiden | Leiden/Louvain | 细胞聚类算法 |
| 分辨率 | 0.5 | 0.1-2.0 | 聚类分辨率，越大聚类越多 |

## 参数说明

### 质控过滤参数详解

#### 基因数过滤

- **最小基因数**: 过滤检测基因过少的细胞，通常是死细胞或空液滴
- **最大基因数**: 过滤检测基因过多的细胞，可能是双细胞或多细胞

**推荐值**:
- 10x数据: min=200-500, max=5000-7000
- Smart-seq数据: min=1000-2000, max=8000-10000

#### 线粒体比例过滤

- **原理**: 线粒体基因比例高通常表示细胞状态差或凋亡
- **推荐值**:
  - 人样本: 10-20%
  - 小鼠样本: 5-15%

### 聚类参数详解

#### 分辨率选择

| 分辨率 | 聚类数量 | 适用场景 |
|-------|---------|---------|
| 0.1-0.3 | 较少(3-5) | 大的细胞类型 |
| 0.4-0.6 | 中等(5-15) | 常规分析 |
| 0.7-1.0 | 较多(10-20) | 细胞亚群 |
| 1.0+ | 很多(20+) | 稀有亚群 |

## 常见问题

### 安装问题

**Q: pip安装依赖失败？**
```
A: 尝试以下方法：
1. 升级pip: python -m pip install --upgrade pip
2. 使用国内镜像: pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
3. 逐个安装依赖
```

**Q: Streamlit启动报错？**
```
A: 检查以下几点：
1. Python版本是否为3.10-3.12
2. 是否在正确的目录
3. 依赖是否完整安装
```

### 数据问题

**Q: 文件上传失败？**
```
A: 可能原因：
1. 文件格式不正确，确保是有效的.h5ad或.zip文件
2. 文件过大，超过1GB限制
3. 文件损坏，尝试重新生成
```

**Q: 内存不足？**
```
A: 解决方法：
1. 启用降采样功能
2. 减少分析的基因数量
3. 增加系统内存
```

### 分析问题

**Q: 聚类结果不理想？**
```
A: 调整建议：
1. 尝试不同的分辨率参数
2. 检查质控是否过于严格或宽松
3. 增加PCA主成分数量
```

**Q: 富集分析结果为空？**
```
A: 可能原因：
1. 差异基因太少，放宽阈值
2. 数据库连接失败，检查网络
3. 基因名称格式不匹配
```

## 最佳实践

### 数据分析流程建议

```
1. 数据导入
   └─ 检查数据基本信息
   └─ 确认细胞和基因数量合理

2. 质控设置
   └─ 先用默认参数
   └─ 根据小提琴图调整阈值
   └─ 不要过度过滤

3. 分析执行
   └─ 使用默认参数快速分析
   └─ 根据结果调整参数
   └─ 记录关键参数

4. 结果解读
   └─ 检查聚类质量
   └─ 验证标记基因
   └─ 结合生物学知识

5. 导出报告
   └─ 保存所有参数配置
   └─ 导出完整分析报告
```

### 参数调整策略

| 情况 | 调整方向 |
|-----|---------|
| 细胞数过少 | 放宽质控阈值 |
| 聚类不清晰 | 调整分辨率 |
| 聚类过多 | 降低分辨率 |
| 标记基因不显著 | 调整差异分析阈值 |
