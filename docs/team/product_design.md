## 一、任务概述

### 1.1 目标
将LocalSingleCell（本地化单细胞&空间转录组分析工具）从Streamlit默认风格改造成类似NCBI简洁优雅风格的本地面向用户的交互网页。

### 1.2 设计原则
- 简洁优雅，专业的生物信息学工具风格
- 用户在每步分析时可选择参数调整，否则用默认参数（最大众选择）
- 保持完全本地化
- 参考NCBI (https://www.ncbi.nlm.nih.gov/) 设计风格

## 三、设计方案

### 3.1 色彩方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #2E5A88 | NCBI深蓝色 - 主按钮、导航 |
| 浅蓝 | #4A7AB3 | 悬停状态 |
| 深蓝 | #1E3F5E | 强调文字 |
| 成功 | #2D7D46 | 绿色 - 成功状态 |
| 警告 | #D4A017 | 金色 - 警告信息 |
| 错误 | #B33030 | 红色 - 错误状态 |
| 背景 | #FFFFFF | 主背景 |
| 次级背景 | #F5F7FA | 卡片区域 |
| 侧边栏 | #E8ECF1 | 三级背景 |
| 边框 | #D1D5DB | 边框颜色 |

### 3.2 字体方案

| 用途 | 字体 | 大小 | 字重 |
|------|------|------|------|
| 主标题 | Inter | 32px | 700 |
| 副标题 | Inter | 24px | 600 |
| 章节标题 | Inter | 18px | 600 |
| 正文 | Inter | 15px | 400 |
| 参数代码 | JetBrains Mono | 13px | 400 |

### 3.3 导航结构重组

**原结构（8-10个入口）：**
- 首页、数据导入、AI分析、分析流程配置、结果可视化、空间分析流程、空间可视化、基因富集、结果导出、帮助

**新结构（6个核心入口）：**

| 新入口 | 包含原有模块 | 说明 |
|--------|--------------|------|
| 🏠 首页 | 首页 | 欢迎页 + 快速开始向导 |
| 📁 数据管理 | 数据导入 | 数据导入 + 当前数据状态 |
| 🔬 分析工具 | 分析流程配置 + AI分析 + 基因富集 | 一站式分析配置 |
| 📊 结果查看 | 结果可视化 + 空间可视化 | 整合可视化展示 |
| 💾 导出 | 结果导出 | 数据导出与报告生成 |
| ❓ 帮助 | 帮助文档 | 使用指南与FAQ |

### 3.4 参数配置优化

**核心原则**：默认优先，自定义按需

```
每步分析参数：
┌─────────────────────────────────────────┐
│ 参数项名称                    [?] 默认值 │
├─────────────────────────────────────────┤
│ 用户看到：默认值 + 快速选项              │
│ → 无需修改 → 直接进入下一步              │
│ → 需要修改 → 点击展开自定义面板          │
└─────────────────────────────────────────┘
```

## 四、页面设计详情

### 4.1 首页
- 4个功能卡片入口（大图标+简短描述）
- 当前分析状态指示
- 快速搜索帮助文档

### 4.2 数据管理
- 顶部显示当前数据状态
- 卡片式导入方式选择
- 高级选项默认折叠

### 4.3 分析工具（核心）
- 步骤式流程设计（质控→降维→聚类）
- 每步提供"使用默认"和"自定义"选项
- 可选：差异分析/富集分析
- AI智能配置入口

### 4.4 结果查看
- 顶部分析结果摘要
- 可视化主展示区
- 右侧统计信息和标记基因

### 4.5 导出页面
- 导出选项列表
- 文件大小预估
- 一键打包下载
