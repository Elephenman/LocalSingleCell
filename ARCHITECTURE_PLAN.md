# NCBI风格架构重构方案

## 1. 现状分析

### 当前架构
- 使用Streamlit单页面应用
- 侧边栏导航（sidebar）
- 页面模块分散在ui/目录
- 状态管理依赖st.session_state

### NCBI网页特点（目标）
1. **顶部导航栏**：全局搜索 + 主要功能入口
2. **工具箱菜单**：分类展示所有分析工具
3. **数据库浏览器**：专业的数据浏览和检索
4. **面包屑导航**：层级式页面结构
5. **工具栏**：常用操作快捷入口
6. **帮助系统**：文档和教程

---

## 2. 新架构设计

### 2.1 页面结构

```
┌─────────────────────────────────────────────────────────────┐
│  🔬 LocalSingleCell    [🔍 搜索工具/基因/数据集...]    [?]  │ ← 顶部导航栏
├─────────────────────────────────────────────────────────────┤
│  首页  |  数据中心  |  分析工具  |  空间分析  |  教程      │ ← 主导航
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────┐  ┌─────────────────────────────────┐  │
│  │                 │  │                                 │  │
│  │   快捷工具面板   │  │        主内容区域               │  │
│  │   (左侧)        │  │                                 │  │
│  │                 │  │                                 │  │
│  └─────────────────┘  └─────────────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 导航结构

| 一级菜单 | 二级菜单 | 功能 |
|---------|---------|------|
| 首页 | - | 欢迎页、状态概览、快速入口 |
| 数据中心 | 数据导入 | h5ad、10x、SRA导入 |
| | 数据浏览 | 当前数据集信息、样本统计 |
| | 数据导出 | 结果导出 |
| 分析工具 | 质控过滤 | QC参数配置、执行 |
| | 降维聚类 | PCA、UMAP、tSNE、聚类 |
| | 差异分析 | 差异基因检测 |
| | 富集分析 | GO、KEGG、Reactome |
| 空间分析 | 空间数据 | Visium数据处理 |
| | 空间可视化 | 空间UMAP、热点分析 |
| AI分析 | 自然语言 | AI命令解析 |
| 教程 | 使用指南 | 文档和教程 |

### 2.3 核心模块设计

```
ui/
├── components/          # 可复用组件
│   ├── navbar.py        # 顶部导航栏
│   ├── toolbar.py       # 工具栏
│   ├── search.py        # 搜索组件
│   ├── dataviewer.py    # 数据浏览器
│   └── breadcrumb.py    # 面包屑
├── pages/               # 页面模块（重构后）
│   ├── home/            # 首页
│   ├── data/            # 数据中心
│   ├── analysis/        # 分析工具
│   ├── spatial/         # 空间分析
│   ├── ai/              # AI分析
│   └── help/            # 帮助
└── styles/              # 样式定制
```

### 2.4 关键技术实现

1. **自定义导航**：使用HTML/CSS构建顶部导航栏
2. **搜索功能**：支持搜索工具、基因名称、数据集
3. **状态管理**：改进的session_state管理
4. **主题定制**：CSS注入实现NCBI风格
5. **响应式布局**：适配不同屏幕尺寸
